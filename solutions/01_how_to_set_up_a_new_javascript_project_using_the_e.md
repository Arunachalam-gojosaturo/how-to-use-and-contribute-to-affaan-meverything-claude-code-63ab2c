# How to set up a new JavaScript project using the everything-claude-code repository as a starting point

_Clone the repository, install dependencies, and start a new project_

## Steps

1. Open a terminal and navigate to the directory where you want to create your new project using the command `cd ~/projects`
2. Clone the everything-claude-code repository using the command `git clone https://github.com/affaan-m/everything-claude-code.git`
3. Navigate into the cloned repository using the command `cd everything-claude-code`
4. Install the required dependencies using the command `npm install` or `yarn install`
5. Create a new branch for your project using the command `git checkout -b my-new-project`

## Pitfalls

- Forgetting to install dependencies can lead to errors when running the project
